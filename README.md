## Getting Started

1. Clone the repository to your local machine.
```sh
git clone git@github.com:GravityCodes/blog-admin-end.git
```
2. Navigate to the project directory.
```sh
cd blog-admin-end
```
3. Install the required dependencies.
```sh
npm install
```
4. setup environment variables.
```sh
cp .env.example .env
```
5. Start the server.
```sh
npm run dev
```
